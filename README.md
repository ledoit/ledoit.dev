# Portfolio (Nuxt)


This folder contains a minimal Nuxt scaffold for the portfolio.

Quick start:

```bash
# install deps
npm install

# run dev server
npm run dev
```

Notes:
- The Nuxt app loads `assets/styles.scss` (configured in `nuxt.config.ts`).
- I removed the old root `index.html` and `styles.scss` to avoid confusion â€” the Nuxt app now provides the site.
- If you want to watch SCSS independently, run `npm run sass:watch` after `npm install`.
